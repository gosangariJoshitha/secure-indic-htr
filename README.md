# 🚀 Project Overview

SecureDocAI is an AI-powered secure document digitization platform designed to convert printed and handwritten documents into editable digital documents while preserving the original layout.

Unlike conventional OCR applications, SecureDocAI combines **Artificial Intelligence**, **Document Intelligence**, and **Cybersecurity** into a unified platform.

It supports:
- Telugu
- Hindi
- English
- Printed Documents
- Handwritten Documents
- Images
- PDFs
- Camera Scans

# ⚙️ Installation

```bash
git clone https://github.com/gosangariJoshitha/secure-indic-htr.git
cd secure-indic-htr

python -m venv .venv

# Windows
.venv\Scripts\activate

pip install -r requirements.txt

streamlit run app.py
```
